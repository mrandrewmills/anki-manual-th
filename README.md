## লেখার নির্দেশিকা

২০২৪ সালে, আমরা ডকুমেন্টেশনে ব্যবহৃত ফর্ম্যাটিংকে মানসম্মত করার প্রচেষ্টা শুরু করি। আপনি হয়তো দেখতে পাবেন যে বর্তমান ম্যানুয়ালটি এখনও এই সমস্ত নির্দেশিকা সম্পূর্ণরূপে মেনে চলে না। তবে, আমরা আমাদের লেখায় এই নির্দেশিকাগুলি মেনে চলব। আপনি যদি কোনও সংশোধন বা আপডেট জমা দেন, তাহলে অনুগ্রহ করে এই নির্দেশিকাগুলি অনুসরণ করুন, এবং আপনি আপনার বিদ্যমান বিষয়বস্তুগুলিকেও সেগুলি মেনে চলতে পারেন।

- একক উদ্ধৃতি ```এর পরিবর্তে`"` ডাবল উদ্ধৃতি ব্যবহার করুন।
- কোনও শব্দকে জোর দেওয়ার জন্য, যেমন সংজ্ঞায়, তির্যক `_` ব্যবহার করুন।
- কোনও শব্দকে নিজেই উল্লেখ করার সময় তির্যক ব্যবহার করুন, যেমন, "_dog_ বা _cat_ অনুসন্ধান করতে, `dog or cat` ব্যবহার করুন"।
- UI এর কিছু অংশ উল্লেখ করার সময়, বোল্ড `**` ব্যবহার করুন।
- কীবোর্ড বোতামগুলি উল্লেখ করার সময়, সেগুলিকে `<kbd></kbd>` এ মুড়িয়ে দিন।
- কমান্ড বা ইনপুট চিত্রিত করার সময়, সেগুলিকে `` ``` ব্যাকটিক্সে মুড়িয়ে দিন।

## পুরানো লিঙ্কগুলি বজায় রাখা

### বিষয়গুলি সরানো

আমরা প্রায়শই প্রতিটি পৃষ্ঠায় ব্যবহারকারীদের নির্দিষ্ট বিষয়গুলিতে গাইড করার জন্য অ্যাঙ্কর লিঙ্ক ব্যবহার করি। এই লিঙ্কগুলি বিষয়গুলির উপর তৈরি করা হয়, যেমন, `https://docs.ankiweb.net/deck-options.html#timers`।

যদি আপনি এমন একটি PR জমা দেন যা কোনও বিষয় পরিবর্তন করে বা সরিয়ে দেয়, তাহলে দয়া করে একটি লুকানো অ্যাঙ্কর যোগ করুন। মূল লিঙ্কটি রাখতে কাজ করছে:

উদাহরণ: "ঘড়ি" শিরোনামটি "টাইমার" এ পরিবর্তন করতে, প্রতিস্থাপন করুন:

```
## ঘড়ি
```

এর সাথে:

```
## টাইমার
<a id="ঘড়ি"></a>
```

### একটি পৃষ্ঠা মুছে ফেলা

যদি আপনি এমন একটি PR জমা দেন যা ম্যানুয়াল বা FAQ থেকে একটি পৃষ্ঠা সম্পূর্ণরূপে সরিয়ে দেয়:

1. ম্যানুয়াল [অথবা FAQ: [SUMMARY](../../../faqs/blob/main/src/SUMMARY.md)] এর [SUMMARY](src/SUMMARY.md) থেকে সেই "অধ্যায়"টি মুছে ফেলুন।

2. ডকুমেন্টের একটি নতুন অবস্থানে পুনঃনির্দেশিত করবেন কিনা তা বিবেচনা করুন নাকি এটিকে [404 ত্রুটি](https://docs.ankiweb.net/404.html) হিসাবে রেখে দেবেন কিনা তা বিবেচনা করুন। পুনঃনির্দেশনা প্রয়োজন হলে, `[output.html.redirect]` বিভাগে একটি এন্ট্রি যোগ করুন [book.toml](book.toml) ম্যানুয়ালটিতে [অথবা FAQ: [book.toml](../../../faqs/blob/main/book.toml)]

- উদাহরণ: `"old-page-name.html" = "new-page-name.html"`
- আরও তথ্যের জন্য [mdBook ডকুমেন্টেশন](https://rust-lang.github.io/mdBook/format/configuration/renderers.html#outputhtmlredirect) দেখুন।
