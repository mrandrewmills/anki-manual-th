# আঙ্কি ম্যানুয়াল-এ অবদান

## স্থানীয় উন্নয়ন

আঙ্কি-ম্যানুয়ালটি [mdBook](https://rust-lang.github.io/mdBook/) ব্যবহার করে তৈরি করা হয়েছে, যা মার্কডাউন ব্যবহার করে ওয়েবসাইট তৈরির জন্য একটি রাস্ট-ভিত্তিক টুল।

### প্রয়োজনীয় ইনস্টলেশন

mdBook ব্যবহার করার জন্য, প্রথমে নিশ্চিত করুন যে আপনার মেশিনে [Rust](https://www.rust-lang.org/tools/install) ইনস্টল করা আছে।

তারপর, mdBook এবং নিম্নলিখিত প্রি-প্রসেসরগুলি ইনস্টল করুন:

```
cargo install mdbook
cargo install mdbook-toc
cargo install mdbook-linkcheck
```

### ডেভেলপমেন্ট এবং প্রিভিউ

স্থানীয়ভাবে anki-manual চালানোর জন্য: কমান্ড দিয়ে বইটি তৈরি করে শুরু করুন:

```
mdbook build
```

তারপর, anki-manual চালান এবং কমান্ড দিয়ে ফলাফল খুলুন:

```
mdbook serve --open
```

এখন আপনি `src/` ফোল্ডারে Markdown ফাইল (`*.md`) সম্পাদনা করতে পারেন এবং আপনার ব্রাউজারের মাধ্যমে রিয়েল-টাইমে পরিবর্তনগুলি দেখতে পারেন।
