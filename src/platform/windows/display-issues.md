# विंडोज प्रदर्शन समस्याएं

<!-- toc -->

विंडोज पर, स्क्रीन पर सामग्री प्रदर्शित करने के तीन तरीके हैं। डिफ़ॉल्ट _सॉफ़्टवेयर_ है, जो धीमा है, लेकिन सबसे संगत है। दो अन्य विकल्प हैं जो तेज़ हैं: _OpenGL_ और _ANGLE_। वे तेज़ हैं, लेकिन वे काम नहीं कर सकते हैं, या प्रदर्शन संबंधी समस्याएं पैदा कर सकते हैं जैसे कि गुम मेन्यूबार, खाली खिड़कियां, और इसी तरह। कौन सा सबसे अच्छा काम करेगा यह आपके कंप्यूटर पर निर्भर करेगा।


## वरीयताएँ स्क्रीन से ड्राइवर बदलना
Anki 23.10+ में, आप **Tools → Preferences** पर जाकर और फिर ड्रॉपडाउन मेनू से ड्राइवर का चयन करके वरीयताएँ स्क्रीन से ग्राफिक्स ड्राइवर बदल सकते हैं।


## कमांड लाइन से ड्राइवर बदलना
यदि आप प्रदर्शन संबंधी समस्याओं का सामना करते हैं, तो आप cmd के माध्यम से सॉफ़्टवेयर मोड में स्विच करने का प्रयास कर सकते हैं:

```bat
echo software > %APPDATA%\Anki2\gldriver6
```

या आप इसे पावरशेल के माध्यम से कर सकते हैं:

```powershell
echo software > $env:APPDATA\Anki2\gldriver6
```

यह कुछ भी प्रिंट नहीं करेगा। फिर आप Anki को फिर से शुरू कर सकते हैं।

डिफ़ॉल्ट व्यवहार पर वापस जाने के लिए, `software` को `auto` में बदलें, या उस फ़ाइल को हटा दें।

## फ़ुल स्क्रीन

Anki 2.1.50+ एक फ़ुल स्क्रीन मोड के साथ आता है, लेकिन विभिन्न समस्याओं के कारण, `OpenGL` का उपयोग करते समय इसे अक्षम करना पड़ा। ऊपर बताए अनुसार सॉफ़्टवेयर रेंडरिंग चालू करने से फ़ुल स्क्रीन विकल्प का उपयोग किया जा सकेगा, हालाँकि कृपया ध्यान रखें कि रेंडरिंग प्रदर्शन प्रभावित हो सकता है।

Anki 23.10+ में, डिफ़ॉल्ट Direct3D ड्राइवर के साथ फ़ुल स्क्रीन मोड समर्थित है।
