# पाठ का आकार

यदि आपको लगता है कि पाठ का आकार गलत है, तो दो पर्यावरण चर हैं जिन्हें आप आजमा सकते हैं:

- ANKI_NOHIGHDPI=1 Qt के कुछ उच्च डीपीआई समर्थन को बंद कर देगा

- ANKI_WEBSCALE=1 Anki के वेब दृश्यों (जैसे डेक सूची, अध्ययन स्क्रीन, आदि) के पैमाने को बदल देगा, जबकि मेनू बार जैसे इंटरफ़ेस तत्वों को अकेला छोड़ देगा। 1 को वांछित पैमाने से बदलें, जैसे कि 1.5 या 0.75।

विंडोज पर आप इन्हें Anki शुरू करना आसान बनाने के लिए एक बैच फ़ाइल में जोड़ सकते हैं। उदाहरण के लिए, अपने डेस्कटॉप पर startanki.bat नामक एक फ़ाइल बनाएं जिसमें निम्नलिखित पाठ हो:

    set ANKI_WEBSCALE=0.75
    start "Anki" "%LocalAppData%\Programs\Anki\anki.exe"

सहेजने के बाद, आप उस सेटिंग के साथ Anki शुरू करने के लिए फ़ाइल पर डबल क्लिक कर सकते हैं।
