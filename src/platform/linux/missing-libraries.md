# गुम पुस्तकालय

यदि Anki शुरू करने में विफल रहता है, तो कृपया इसे `anki` के साथ एक टर्मिनल से चलाएं। यदि यह कहता है कि एक पुस्तकालय गुम है, तो कृपया इसे स्थापित करें और फिर से प्रयास करें।

यदि यह शिकायत करता है कि कोई प्लेटफ़ॉर्म उपलब्ध नहीं है, तो कृपया Anki को निम्नलिखित कमांड लाइन के साथ शुरू करें, जिससे एक गुम पुस्तकालय का पता चलना चाहिए:

```shell
QT_DEBUG_PLUGINS=1 anki
```

apt-get या इसी तरह के साथ पुस्तकालय स्थापित करने के बाद, प्रक्रिया को दोहराएं। सभी आवश्यक पुस्तकालयों को स्थापित करने से पहले आपको इसे कुछ बार करने की आवश्यकता हो सकती है।
