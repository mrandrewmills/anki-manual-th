# ฟิลด์

<!-- toc -->

ฟิลด์คือส่วนต่างๆ ของโน้ตใน Anki เช่น คำถาม คำตอบ หรือข้อมูลเพิ่มเติม คุณสามารถเพิ่ม ลบ หรือจัดเรียงฟิลด์ใหม่ได้ในประเภทโน้ตแต่ละประเภท

## การเพิ่มฟิลด์

ในการเพิ่มฟิลด์ใหม่ ให้เปิดตัวแก้ไขประเภทโน้ต คลิก "จัดการฟิลด์" แล้วเลือก "เพิ่ม" จากนั้นตั้งชื่อฟิลด์ใหม่

## การลบฟิลด์

หากต้องการลบฟิลด์ ให้เลือกฟิลด์ที่ต้องการในหน้าต่าง "จัดการฟิลด์" แล้วคลิก "ลบ" โปรดระวัง: การลบฟิลด์จะลบข้อมูลในฟิลด์นั้นออกจากโน้ตทุกใบที่ใช้ประเภทโน้ตนี้

## การจัดเรียงฟิลด์ใหม่

คุณสามารถเปลี่ยนลำดับของฟิลด์ได้โดยลากฟิลด์ในหน้าต่าง "จัดการฟิลด์" ลำดับของฟิลด์จะมีผลต่อการจับคู่ฟิลด์ขณะนำเข้าไฟล์ข้อมูล

## การใช้ฟิลด์ในเทมเพลต

ในเทมเพลตบัตร คุณสามารถแทรกฟิลด์โดยใช้รูปแบบ `{{ชื่อฟิลด์}}` เช่น

```html
{{Front}} {{Back}}
```

## ฟิลด์จัดเรียง

ฟิลด์จัดเรียง (Sort Field) คือฟิลด์ที่ใช้สำหรับเรียงลำดับโน้ตในหน้าต่างเบราว์เซอร์ คุณสามารถเลือกฟิลด์จัดเรียงได้ในหน้าต่าง "จัดการฟิลด์"

## ฟิลด์แท็ก

คุณสามารถกำหนดให้ฟิลด์ใดเป็นฟิลด์แท็กได้ เพื่อให้ Anki
